Link Best Practices

As with all of the code on your page, how you write it can affect whether or not your page is usable and accessible. There are some basic things to keep in mind with links.

  • The text inside of your link should tell the user what will happen when they click on it
    • Link text should never say something like "click here"
    • Bad: Click here to submit our form
    • Good: You can contact us by submitting a message through our form
  • Test your links to make sure they work, from every page
  • Use the appropriate type of link when adding one to your page

About Navigation

Navigation is coded using an unordered list because navigation is a list of web pages. Relative hyperlinks are used when coding navigation. By default, this creates a vertical list with bullets, which can be styled later for horizontal layout.

About Links

You will be learning about and creating six types of hyperlinks:

  • absolute
  • relative
  • fragment
  • email
  • telephone
  • relative with fragment

Absolute Hyperlinks

Absolute hyperlinks are used to link to another website. For example, visit asu.edu to learn more.

Relative Hyperlinks

Relative hyperlinks are used for pages within your own website. Navigation links are usually relative because they point to other pages in the same site.

E-Mail Hyperlinks

Email hyperlinks will automatically launch the user's mail program. For practice, email me at youremail@asu.edu.

Telephone Number Hyperlinks

If clicked on a smartphone, this link will dial the number: (555) 555-5555.

Fragment Hyperlinks

Fragment hyperlinks jump to a specific part of the same page. For example, click here to go back to the top of this page.

Combining Relative and Fragment Hyperlinks

Relative and fragment links can be combined to link to a specific section on another page. For example: Contact Form.
